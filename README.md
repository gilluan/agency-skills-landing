# Agency Skills Landing Page

> Landing page de alta conversão para o produto "Claude Code Agency Skills"

**URL:** [Em breve no Cloudflare]  
**Stack:** Next.js 14 + Tailwind CSS  
**Deploy:** Cloudflare Pages

---

## 🚀 Deploy

```bash
# Instalar dependências
npm install

# Desenvolvimento
npm run dev

# Build para produção
npm run build
```
